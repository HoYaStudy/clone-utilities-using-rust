# Clone Utilities Using Rust

This project is to clone GNU utilities using Rust

## Utility List

* `true` / `false`
* `echo`
  * -n
* `cat`
  * -n, -b
* `head`
  * -n, -c
* `wc`
  * -l, -w, -c, -m
* `uniq`
  * -c
* `find`
  * -n, -t
* `cut`
  * -d, -f, -b, -c
* `grep`
  * -i, -r, -c, -v
* `comm`
  * -1, -2, -3, -i, -d
* `tail`
  * -n, -c, -q
* `cal`
  * -m, -y
* `ls`
  * -l, -a
